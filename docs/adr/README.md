# ADR Index

This directory stores architecture decision records for platform changes that
alter:

- truth layers,
- validation semantics,
- orchestration boundaries,
- publication rules,
- package-contract expectations.

These ADRs are intentionally lightweight. Their purpose is to stop major
process and architecture changes from becoming silent drift.
