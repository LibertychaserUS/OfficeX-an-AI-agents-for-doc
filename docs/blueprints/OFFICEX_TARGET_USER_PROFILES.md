# OfficeX Target User Profiles

## Current Focus

The first high-value user groups are:

- internet companies
- technology companies
- government and internal public-sector document teams

## Why These Profiles Matter

These users tend to need:

- strict formatting
- citation and source discipline
- reusable templates
- internal review loops
- repeatable formal deliverables
- auditability

## Product Consequences

These profiles increase the importance of:

- materials and citation tracking
- conservative approval defaults
- deterministic mutation
- Word compatibility
- traceability and replay

## Expansion Rule

Broader document domains may be added later, but the first OfficeX core should
optimize for high-constraint internal document production rather than generic
writing assistance.
